## 아마존 웹 서비스와 클라우드

아마존 웹 서비스는 클라우드 컴퓨팅 하면 대표적인 기업임.

아마존 웹 서비스를 통해서 클라우딩 컴퓨팅이 무엇인가 이해하는게 목표.

클라우딩 컴퓨팅은.. 구름위에 설치되있는 컴퓨터를 이용해 실제 작업을 처리. (구름=인터넷)

클라우딩 컴퓨팅은 유행어다.. 인터넷에 연결되있는 거대한 컴퓨터를 사용한다.

[Amazon Web Services](http://aws.amazon.com/ko/)

아마존은 여러가지 상품들이 있다.
